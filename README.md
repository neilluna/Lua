# Lua
Version 0.0.0

Lua environments on Windows. A set of scripts and proceedures to build and use different versions of Lua on Microsoft Windows.

Download Lua from the following places:

1. The directory that you cloned this repository into is referred to as the "installation directory" in this document.
    For example, if you cloned as follows:

    `git clone https://github.com/neilluna/Lua.git C:\Software\Lua`

    This cloned the repository into `C:\Software\Lua`. The `Lua` subdirectory was created as part of the `git clone` operation. `C:\Software\Lua` is the installation directory.

1. Download and install "Build Tools for Visual Studio 2026" from Microsoft: e.g. `vs_BuildTools.exe`

1. Download the Lua 5.1.5 source archive from [lua-5.1.5.tar.gz](https://www.lua.org/ftp/lua-5.1.5.tar.gz). Extract it into the installation directory (e.g. `C:\Software\Lua`).

1. Download the Lua 5.4.8 source archive from [lua-5.4.8.tar.gz](https://www.lua.org/ftp/lua-5.4.8.tar.gz). Extract it into the installation directory (e.g. `C:\Software\Lua`).

1. Download the LuaRocks 3.13.0 release archive from [luarocks-3.13.0-windows-64.zip](https://luarocks.github.io/luarocks/releases/luarocks-3.13.0-windows-64.zip). Extract it into the installation directory (e.g. `C:\Software\Lua`). This should create a subdirectory named `luarocks-3.13.0-windows-64`.
